**О чем этот проект**

В данный момент я собираю бэкенд проекта на Node.js
На данный момент реализовано:
* Регистрация
* Вход
* Работа с бозой данных MongoDB
* Добавлять новые карточки в базу данных MongoDB
* Удалять карточки
* Добавление/удаление к карточкам лайков

* Node.js
